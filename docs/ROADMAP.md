# Product Roadmap

## Phase 1 — Foundation

- Choose the web application stack and package manager.
- Define lesson, question, answer, attempt, and mastery data models.
- Add formatting, linting, tests, and continuous integration.
- Create the application shell and domain navigation.

## Phase 2 — Learning loop

- Build lesson pages and objective navigation.
- Add flashcards and spaced-review scheduling.
- Add quizzes with answer explanations.
- Store local learner progress.

## Phase 3 — Exam readiness

- Add timed mixed-domain exams.
- Add performance-based scenario exercises.
- Weight recommendations using SY0-701 domain percentages.
- Surface weak objectives and readiness trends.

## Phase 4 — Quality and release

- Review content against the latest official objectives.
- Add accessibility and responsive-layout testing.
- Add backup/export controls for learner data.
- Complete security, privacy, and deployment reviews.
